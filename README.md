## Setup

Use the adonis command to install the blueprint

```bash
adonis new yardstick --api-only
```

Init

```bash
adonis serve --dev
```

or manually clone the repo and then run `npm install`.

### Migrations

Run the following command to run startup migrations.

```js
adonis migration:run
```
